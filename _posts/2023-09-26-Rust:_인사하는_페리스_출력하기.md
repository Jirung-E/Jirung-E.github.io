## 인사하는 페리스 출력하기
[공식 문서](https://docs.rs/ferris-says/latest/ferris_says/fn.say.html)

### 코드 작성
```rust
fn main() {
    use ferris_says::*;
    use std::io::{ stdout, BufWriter };
    
    let stdout = stdout();
    let out = "안녕 Jirung!";
    let width = 7;
    
    let writer = BufWriter::new(stdout.lock());
    say(out, width, writer).unwrap();
}
```

### Dependency 추가
```toml
[dependencies]
ferris-says = "0.3.1"
```
버전을 잘 맞춰야 동작합니다.

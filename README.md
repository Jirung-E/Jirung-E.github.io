## 포스팅 방법
command line:
```
./makepost post1.md post2.md
```


### 포스트 작성법
```md
# 제목
요약
- Category
+ tag1
+ tag2
+ tag3
+ tag3

## 본문을 시작합니다.
### 본문1
내용입니다.

### 본문2
본문2의 내용입니다.
엔터도 제대로 쳐줘야합니다.
```
제목과 카테고리는 필수, 요약과 태그는 선택
